# Runner Game Using Pygame

This is a simple runner game developed using the Pygame library in Python. The player controls a character that runs continuously, and the objective is to avoid obstacles (snails and birds) and collect points.

## Features

- Continuous running character
- Jumping mechanics to avoid obstacles
- Scoring system based on time survived
- Simple graphics and animations

## Requirements

- Python 3.x
- Pygame library
  
## Installation

1. Install Python from [python.org](https://www.python.org/).
2. Install Pygame using pip:

   ```
   pip install pygame
   ```

3. Clone this repository.
4. Navigate to the project directory.
5. Run the game:

   ```
   python main.py
   ```

6. Enjoy playing the game!

## Controls

- Press the spacebar to make the character jump.
